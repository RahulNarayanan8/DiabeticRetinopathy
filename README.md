# DiabeticRetinopathy
ML for predicting severity of diabetic retinopathy from eye scans.


# Getting data

Download data from this link:
https://www.kaggle.com/datasets/sovitrath/diabetic-retinopathy-224x224-2019-data/data

Rename the folders to the following (for training and class label purposes):
0_No_DR
1_Mild
2_Moderate
3_Severe
4_Proliferate_DR
